# Synclaw Core (OSS)

This track documents the open-source `core` edition.

## Includes

- Manual/BYO OpenClaw setup
- Workspace/team/task/document/chat workflows
- GitHub/Google auth support

## Excludes

- Managed provisioning and assisted launch operations
- Billing and paid support flows
- Managed gateway control-plane automation

Use `SYNCLAW_EDITION=core` and `NEXT_PUBLIC_SYNCLAW_EDITION=core`.
