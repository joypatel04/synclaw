# Self-hosted Prerequisites

Self-hosted is intended for developers and technical teams.

## Required accounts and services

- GitHub account for source access
- Convex account and project
- OpenClaw gateway access
- MCP server deployment target
- OAuth app credentials for auth providers

## Required local tools

- Bun or Node.js runtime
- Git
- Shell access to set env variables

## Required product knowledge

- Convex deployments and environment management
- OAuth callback setup
- Application deploy workflows
- Basic operational monitoring and troubleshooting

## Important expectation

Self-hosted is not "hard on purpose."  
It is naturally complex because infra ownership stays with you.
